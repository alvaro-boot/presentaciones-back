Rutas de monitoreo en data/rutas-supervision.json: array rutas, cada una con coords [[lat,lng], …], opcional paradas [{ lat, lng, nombre } o [lat,lng,"nombre"]], nombre, color opcional, cicloSegundos (duración del recorrido), faseInicial 0–1 para desfasar motos. Estilo global en config-mapa.json → supervision. En data/puntos-pereira.json: conjunto (rojo, id y orden para el perímetro entre rojos). porteria / ronda (azul, conjuntoId = id del conjunto): con 3 o más azules del mismo conjunto se cierra un polígono y se rellena con color claro; con 2, solo línea entre ellos; con 1, línea al rojo. El nombre se muestra junto al punto solo en portería y conjunto; los puntos ronda solo muestran el círculo (el nombre sigue en el popup al hacer clic). Opcional en cada punto: color en hexadecimal (#dc2626). Si no va, se usan los colores por tipo del archivo de configuración. Editor: los cambios se guardan en este navegador (localStorage: puntos y rutas por separado). Use Descargar JSON para copiar puntos-pereira.json o Descargar JSON (rutas) para rutas-supervision.json. Solo archivo del servidor / Solo archivo rutas del servidor descartan la copia local de cada uno.
